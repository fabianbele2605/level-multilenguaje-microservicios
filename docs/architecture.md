## ๐ Flujo de Datos

### Flujo Principal (Java โ Go โ Rust โ Python)
1. **Cliente** envรญa request a Java:8081
2. **Java** procesa y envรญa a Go:8080
3. **Go** coordina y envรญa a Rust:8001
4. **Rust** valida datos y envรญa a Python:8000
5. **Python** analiza y responde
6. **Respuesta** viaja de vuelta: Python โ Rust โ Go โ Java โ Cliente

### Flujo de Cรกlculos Pesados (Python โ C++)
1. **Python** recibe datos validados
2. **Python** envรญa a C++:8002 para cรกlculos intensivos
3. **C++** procesa y devuelve resultados
4. **Python** combina anรกlisis bรกsico + cรกlculos pesados
